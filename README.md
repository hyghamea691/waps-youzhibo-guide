# waps-youzhibo-guide

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放和展示多个独立的 HTML 页面。这些页面各自独立，不依赖于特定域名或网站，仅作为静态资源存档与分发使用。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── page1.html
│   └── page2.html
├── assets/             # 存放页面所需的静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：所有独立 HTML 页面均放置在此目录下。
- **assets/**：页面引用的外部资源文件，按类型分类存放。
- **index.html**：可选的入口页面，用于列出或导航至各独立页面。

## 页面归档说明

- 每个 HTML 页面均为功能完整的独立文件，可直接通过浏览器打开。
- 页面之间无相互依赖，可以单独使用或修改。
- 归档页面仅用于展示和参考，不提供任何形式的服务或交互功能。

## 维护说明

- 如需新增页面，请在 `pages/` 目录下创建新的 HTML 文件，并确保其引用资源路径正确。
- 修改或更新页面时，请保持文件结构清晰，并同步更新本 README 文件（如有必要）。
- 本仓库仅用于静态资源托管，不包含任何后端逻辑或动态内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件。
3. 或通过 GitHub Pages 等服务托管后访问。
